# GEO Performance Report Dashboard

An interactive, responsive client report dashboard built with HTML, CSS, and Chart.js to track Generative Engine Optimization (GEO) performance metrics.

## Features

- **Multi-page Navigation**: A fully styled sidebar navigation allowing the user to switch seamlessly between different reporting pages:
  - Cover / Executive Summary
  - Scorecard Overview
  - GEO Traffic Distribution
  - Share of Voice (SOV)
  - Keyword Performance
  - Recommendations
- **Interactive Visualizations**: Powered by Chart.js for beautiful charts representing data points.
- **Premium Design System**: Modern typography (Space Grotesk & Inter), dynamic hover animations, and a curated professional color palette.
- **Responsive Layout**: Designed to look great across multiple viewport resolutions.

## How to View Locally

Simply open the file directly in any modern web browser:

```bash
open index.html
```

or serve it using a lightweight local HTTP server (like Python's built-in module):

```bash
python3 -m http.server 8000
```
Then visit `http://localhost:8000` in your browser.
